### About This Project
This is a personal educational project developed in my spare time outside of university studies. My goal isn’t just to build a game, but to gain practical, low-level experience with Reinforcement Learning (RL) algorithms.

**What I am focusing on:**

* **C++ Beyond Theory:** Instead of using existing game engines, I manually implement the game logic and memory management to deepen my understanding of systems programming.
* **"Under the Hood" Mathematics:** I build the basic structures for the Deep Q-Network (DQN) from scratch to see how mathematical formulas translate into functional code.
* **Active Experimentation:** I tune hyperparameters (learning rate, discount factor, etc.) and observe how they impact the agent's behavior in real-time.
* **Continuous Iteration:** This project is a work in progress. I started with basic mechanics and am now shifting my focus toward training stability and computational optimization.
